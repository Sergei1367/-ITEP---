# ITEP — Industrial Trust Edge Platform

ITEP — это платформа, которая превращает производственные действия в проверяемые, безопасные и подтверждённые события на уровне edge.

## Проблема

В промышленности отсутствует доверие к данным:

* события нельзя однозначно доказать
* логи можно изменить
* изображения не являются доказательством
* облако не всегда доступно
* AI не может принимать safety-критичные решения самостоятельно

## Решение

ITEP создаёт доверенный слой производственных фактов:

* edge-first архитектура
* safety-aware принятие решений
* evidence и audit trail
* работа без зависимости от облака
* отложенная синхронизация

## Архитектура

ITEP построен вокруг трёх контуров:

* Safety Loop
* Decision / Event Loop
* Evidence / Trust Loop

А также минимального execution-атома:

Observation → Decision → Safety → Human → Evidence → Sync
